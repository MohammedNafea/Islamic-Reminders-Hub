# المصحف الشريف والتفسيرات

## المصادر المعتمدة في الموقع

| المصدر | API Endpoint | الاستخدام |
|--------|-------------|-----------|
| النص العثماني | `/v1/surah/{n}/ar.alafasy` | عرض الآيات مع الصوت |
| تفسير الجلالين | `/v1/surah/{n}/ar.jalalayn` | التفسير المختصر للمحققين |
| التفسير الميسر | `/v1/surah/{n}/ar.muyassar` | التفسير المبسط المبني على الأدلة |

## الارتباط بتطبيق Islamic Reminders Hub

صفحات السور مرتبطة بتطبيق Islamic Reminders Hub عبر:

- `/quran` — صفحة المصحف الشريف الرئيسية
- `/quran/{surahNumber}` — سورة محددة مع تفسيراتها
