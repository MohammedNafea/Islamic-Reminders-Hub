# تحديث الحزم والإضافات

تم تثبيت حزم الواجهة الأمامية وإضافتها إلى `package.json` الرئيسي الخاص ببيئة العمل (workspace) لدعم نظام التصميم وتخصيص المكوّنات.

- `tailwindcss`
- `postcss`
- `autoprefixer`
- `class-variance-authority`
- `clsx`
- `tailwind-merge`
- `lucide-react`

تمت عملية التثبيت بنجاح وتحديث `pnpm-lock.yaml`.
