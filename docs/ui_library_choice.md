# اختيار مكتبة واجهة المستخدم (UI Library)

تم اختيار **Tailwind CSS** مدمجة مع **shadcn/ui** كحل أمثل لتصميم واجهات المشروع لعدة أسباب:

1. **مرونة عالية**: `shadcn/ui` توفر مكونات يمكنك نسخها ولصقها والتعديل عليها بحرية تامة باستخدام كلاسات Tailwind دون قيود كباقي المكتبات المدمجة.
2. **تصميم حديث**: تتوافق بشكل كامل مع المعايير الجمالية الحديثة، بما فيها Dark Mode وأسلوب Glassmorphism.
3. **أداء ممتاز**: لا تضيف أي عبء إلى الحزمة النهائية (bundle size) لأنك تمتلك كود المكوّن نفسه.
4. **دعم للميزات الإضافية**: استخدام حزم مثل `clsx` و `tailwind-merge` يسهل إدارة الفئات (classes) الديناميكية.

يمكن الاطلاع على توثيق المكتبات الرسمية:
- [Tailwind CSS](https://tailwindcss.com/docs)
- [shadcn/ui](https://ui.shadcn.com/docs)
